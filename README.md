# Window List Zoom

## Description

A Cinnamon extension that adds a hover zoom effect to grouped window list icons. When you hover over a window list item, it smoothly scales up under the cursor, then shrinks back when you move away.

## Compatibility

This extension has been tested to be compatible with Cinnamon 6.4.

## Settings

- **Zoom scale factor**: How much window list icons grow when you hover over them (1.0x–2.0x, default 1.3x)

## Installation

1. Copy the extension folder to `~/.local/share/cinnamon/extensions/wl-zoom@mubashirzamir`
2. Reload Cinnamon (`Alt+F2` → `r` → `Enter`, or log out/in)
3. Enable it in **Cinnamon Settings → Extensions**
4. Click the gear icon next to the extension to adjust the zoom factor
