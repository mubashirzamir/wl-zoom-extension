# Panel Zoom Effect

## Description

A Cinnamon extension that adds a macOS dock-style zoom effect to your panel applets. When you hover over the panel, applets smoothly scale up under the cursor, then shrink back down when you move away.

## Compatibility

This extension has been tested to be compatible with Cinnamon 6.4.

## Settings

- **Zoom scale factor**: How much applets grow when you hover over the panel (1.0x–2.0x, default 1.3x)

## Installation

1. Copy the extension folder to `~/.local/share/cinnamon/extensions/panel-zoom@mubashirzamir`
2. Reload Cinnamon (`Alt+F2` → `r` → `Enter`, or log out/in)
3. Enable it in **Cinnamon Settings → Extensions**
4. Click the gear icon next to the extension to adjust the zoom factor
